Challenges and Solutions
Several problems were encountered during the implementation of the project.
Handling Large Dataset
The dataset contained Model Years as early as 1997 which had missing values this paused a challenge when creating dashboard  it would show blank visual especially KPIs. To solve this the model was  optimized  removing unnecessary columns and tables, reducing the dataset size. 
YOY Adoption DAX .The dataset did not provide specific purchase  date at making it difficult to calculate the YOY Adoption Rates.
Map Visualization. Dataset provided Latitudes only without longitude making it difficult to visualize a map and with the provided cities  they  shared names with other cities around the globe or even with the united states itself making data spill over other continents. To solve this we added a drill through which filtered the dataset by only state. Washington.
