Creating Scatter chart for Make 17


  1. GO TO visualization Pane 
  2. Drag the  Scatter chart visualization to the report View Page 
  3. On the Values Field Drag Model then Make (this will be used to create drill- through and the order determine so be keen )
  4. X axis Drag PHEV Distribution
  5. Y Axis drag CAFV Eligibility
  6. Size Drag BEV Distribution(this determine the size of the Bubbles in scatter plot)
  7. Go to General Tab and Edit the Properties and position to adjust the Viz
